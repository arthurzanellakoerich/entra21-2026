# 📚 Entra21 - 2026 | Repositório de Atividades

Este repositório reúne todas as atividades e projetos desenvolvidos durante o programa **Entra21**, uma iniciativa de formação profissional em tecnologia que combina lógica de programação, desenvolvimento back-end com Java e desenvolvimento front-end com HTML, CSS e JavaScript.

---

## 🎯 Sobre o Programa

O **Entra21** é um programa de capacitação profissional voltado para jovens que desejam ingressar na área de tecnologia da informação. Ao longo do programa, os participantes desenvolvem habilidades em:

- Lógica de programação
- Linguagem Java
- Estruturas de dados e algoritmos
- Desenvolvimento web (HTML, CSS, JavaScript)
- Boas práticas de versionamento com Git e GitHub

## ☕ Projetos Java (`/src`)

### 📝 Lista de Exercícios 1 — Fundamentos (`ListaExcercicios/`)

Primeiros exercícios em Java, focados nos fundamentos da linguagem:

- Declaração e uso de variáveis (`int`, `String`, `double`)
- Entrada de dados com `Scanner`
- Operações aritméticas básicas (soma, subtração, multiplicação, divisão)
- Saída formatada com `System.out.println` e `System.out.printf`
- Manipulação de strings e dados do usuário

---

### 📝 Lista de Exercícios 2 — Condicionais (`ListaExcercicios2/`)

Exercícios com foco em estruturas de controle e tomada de decisão:

- Verificação de número par ou ímpar
- Comparação entre dois números
- Identificação de número positivo, negativo ou zero
- Verificação de aprovação/reprovação com base em nota
- Uso de `if`, `else if` e `else`
- Entrada e validação de dados via `Scanner`

---

### 📦 Vetores (`Vetores/`)

Introdução ao conceito de arrays em Java:

- Declaração e inicialização de arrays (`String[]`)
- Acesso e modificação de elementos por índice
- Iteração com `for` indexado
- Iteração com `while`
- Iteração com `for-each` (enhanced for)
- Introdução ao `ArrayList`

## 🌐 Projetos Front-end (`/frontend`)

### 👥 Sistema de Cadastro de Clientes (`cadastro-clientes/`)

Aplicação web completa para gerenciamento de clientes, com:

- **`index.html`** — Página inicial com navegação
- **`cadastro.html`** — Formulário para cadastrar novos clientes
- **`clientes.html`** — Listagem de todos os clientes cadastrados
- **`script.js`** — Lógica de cadastro, armazenamento e listagem usando `localStorage`
- **`style.css`** — Estilização com layout centralizado e design em cards

**Funcionalidades:**
- Cadastro de novos clientes via formulário
- Armazenamento de dados no navegador (localStorage)
- Listagem de todos os clientes cadastrados
- Interface responsiva e amigável

---

### 🎬 Sistema de Cinema — CineMax (`cinema/`)

Simulador de reserva de assentos para cinema, com:

- **`index.html`** — Interface principal com mapa de assentos
- **`script.js`** — Lógica de seleção, ocupação e confirmação de assentos
- **`style.css`** — Layout visual com mapa de sala, tela e legenda

**Funcionalidades:**
- Visualização da sala de cinema com grade de assentos
- Seleção interativa de assentos (livre / selecionado / ocupado)
- Exibição de resumo com assentos escolhidos e valor total
- Confirmação de compra com atualização do estado dos assentos
- Exibe informações do filme: nome, horário e sala

## 🚀 Como Executar

### Projetos Java
1. Abra o projeto no **IntelliJ IDEA**
2. Navegue até a classe desejada dentro de `src/`
3. Chame o método do exercício desejado dentro do `main` e execute

### Projetos Front-end
1. Navegue até a pasta do projeto dentro de `frontend/`
2. Abra o arquivo `index.html` no navegador, ou use a extensão **Live Server** no VS Code
